# TinyDebugLM

## Overview

TinyDebugLM is an Agentic AI Debugging Assistant designed to understand software errors, identify root causes, suggest fixes, and iteratively improve its recommendations using user feedback.

Unlike traditional chatbots, TinyDebugLM acts as a debugging agent capable of analyzing programming errors, environment issues, dependency conflicts, and runtime failures.

---

## Objectives

- Train a custom Tiny Language Model
- Build an Agentic Debugging System
- Create a VS Code Extension
- Deploy using FastAPI
- Support iterative debugging
